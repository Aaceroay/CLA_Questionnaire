Interactive CLA Self-Assessment Tool
Copyright © 2026 Alejandro Acero Ayuda
https://github.com/alejandro-acero-ayuda/cla-questionnaire
 
This interactive questionnaire implements the validated 19-item
Critical Language Awareness (CLA) instrument developed by:
 
    Beaudrie, S., Amezcua, A., & Loza, S. (2019).
    Critical language awareness for the heritage context:
    Development and validation of a measurement questionnaire.
    Language Testing.
    https://doi.org/10.1177/0265532219844293
 
Any use, adaptation, or redistribution of this work must credit
both (1) Alejandro Acero Ayuda as the creator of this
implementation, and (2) Beaudrie, Amezcua & Loza (2019) as the
developers of the underlying measurement instrument.
